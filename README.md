# 说明

使用官方提供的 Open Api 进行开发。

- Ktor Client（CIO Engine）
- Kotlinx lib
    - kotlinx-serialization-json
    - kotlinx-datetime

采用 `toml` 作为配置文件，使用 [ktoml](https://github.com/akuleshov7/ktoml) 解析。
